How to Run a Flask App in Docker

This guide shows you how to build and run a Flask web app inside a Docker container. Below are the Docker setup instructions and screenshots of the app running.

Steps to Run the Dockerized Flask App
  1. Clone the Repository: git clone https://github.com/yourusername/flask-docker-app.git
  2. Navigate to the project folder: cd flask-docker-app
  3. Build the Docker image: docker build -t flask-docker-app .
  4. Run the container: docker run -p 5000:5000 flask-docker-app
  5. Visit http://localhost:5000 in your browser to see the Flask app in action!
