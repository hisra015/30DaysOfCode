Terraform EC2 Instance

Provisioning an AWS EC2 Instance with Terraform

This project uses Terraform to provision an EC2 instance on AWS. Below are the steps to recreate this setup.

Steps to Create the EC2 Instance
  1. Install Terraform on your machine.
  2. Configure your AWS credentials using aws configure.
  3. Create the main.tf file with the Terraform configuration.
  4. Run terraform init to initialize the project.
  5. Run terraform apply to create the EC2 instance.
  6. After verification, run terraform destroy to clean up the resources.
