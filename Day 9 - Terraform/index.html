Provisioning an AWS EC2 Instance with Terraform

This guide documents how I used Terraform to provision an EC2 instance on AWS. Below are the detailed steps and screenshots of the process.

Steps to Create the EC2 Instance
  1. Install Terraform and configure AWS CLI credentials.
  2. Create a `main.tf` file with the necessary Terraform configuration.
  3. Run terraform init to initialize the project.
  4. Run terraform apply to create the EC2 instance.
  5. Verify the instance via the AWS console or CLI.
  6. Run terraform destroy to clean up the resources.
