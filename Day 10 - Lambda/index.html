AWS Lambda Serverless App

Deploying a Serverless App with AWS Lambda

This project demonstrates how to build a serverless application using AWS Lambda, integrated with API Gateway for web access. The infrastructure is deployed using Terraform, automating the setup and deployment of AWS resources. Follow the steps below to recreate this project:

Steps to Create the Lambda Function
  1. Write the Lambda function in Python (save as `lambda_function.py`).
  2. Create a `main.tf` Terraform configuration file to deploy the Lambda function and associated AWS resources.
  3. Run terraform init to initialize the project.
  4. Run terraform apply to create the Lambda function.
  5. Invoke the function using AWS CLI or integrate it with API Gateway for web access.
  6. Use terraform destroy to clean up resources when no longer needed.
