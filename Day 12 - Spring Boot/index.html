Spring Boot Book API

This is a demonstration of my Spring Boot Book Management API.
The API allows users to create, read, update, and delete book records.

  1. Start the Spring Boot server by running mvn spring-boot:run.
  2. Use Postman or cURL to send a POST request to http://localhost:8080/api/books with the book details.
  3. Check that the book was added successfully by sending a GET request to /api/books.
